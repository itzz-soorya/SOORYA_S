# Sabari Sekaran Portfolio

A premium cinematic developer portfolio built using React, Vite, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Features

- Cinematic hero section
- Responsive design
- Interactive stack section
- Featured projects showcase
- Internship experience section
- Resume preview & download
- Smooth animations
- Modern matte black + gold UI

## Projects Included

- AI Leaf Disease Image Captioning
- Planora
- Billing System
- Raliax AC Waiting Hall System

```bash
npm install
npm run dev
